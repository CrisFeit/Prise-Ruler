Introdução

A regua de recompenas (Prise Ruler) serve para o controle de visualização do status de bonus e descontos que o usuário ganha conforme aumenta ou diminui o valor de suas compras

É a partir do valor Total (com ou sem descontos) que é calculado as regras de negócio

Para seu funcionamento, é necessário:

  • Estar ativado no CMS da Vtex.
  • Pelo menos uma regra configurada na régua.
  • Possuir algum item no minicart / sacola.

    • Localização

    A regua é um - Custom Element - com o nome de Prise-Ruler localizada no CMS(Content Management System) da Vtex em :
    https://edbr.myvtex.com/admin/a ou https://edbr.vtexcommercestable.com.br/admin/a

    Na página de CMS selecione:

    1. Custom Elements
    2. Prise-Ruler
    3. Update

    Na página de Custom Element:

    1. Ative a regra que ira subir
    2. selecione edit

    Código Html da Regua

    Cada Section de classe "js--prise-item" é um conjunto de elementos que trocam de acordo com a o valor total do minicart do usuário

    data-limit : valor a ser alcançado para que o usuário ganhe o prêmio *A Section contendo a frase que irá ser mostrada é a qual o valor total do usuário for : maior do que o data-limit da Section anterior menor do que o data-limit da Section atual

    Configuração

    Poderá ser alterado somente os valores dos itens.

    • data-limit : não deve conter nenhum caracter que não seja números
      exemplo: R$ 109,00 data-limit="10900"
    • data-cupom : quando não hover prêmios deixar em brando
      exemplo: data-cupom=""
    • text-content: Frase a ser exibida
      *Deixe o valor do meio sempre 0
      exemplo: < p> class="prise__text-content"> com mais < span class="prise__remain" > 0 < /span> ganhe frete grátis! < /p> < p class="prise__text-content"> com mais < span class="prise__remain" > 0 você ganha um cupom de R$40,00 ! < /p> < p class="prise__text-content prise__text-content--complete"> Você Ganhou Frete Grátis + R$ 40,00 em desconto < /p>
